About Burp HTTP History Browser -

is a Swiss army knife for Burp HTTP history: import, filter, diff, replay, and export, all in the browser.

Built for Burp Suite Community Edition. Runs as a PWA: install it, work offline, and keep your data on your device.

What problem does this solve?

  • Burp Suite Community Edition does not include disk-based projects, so HTTP history, sitemap data, and Logger output are lost when a temporary session ends.
  • CE users can export records (Ctrl+A, then Save items) from Proxy->HTTP history, Target->Sitemap, or Logger to a Burp XML file.
  • better-bhhb opens those exports and presents them in a fast, filterable UI.

Getting data in

  • Burp extension (recommended): install the extension from GitHub Releases, keep this app open, then in Burp use Extensions → Send selected… (or Send all… for proxy history / sitemap). The app imports automatically within a couple of seconds.
  • Manual import: use Import files…, drag-and-drop XML onto the page, or reopen a file from Imported sessions… (recent exports are remembered in IndexedDB).
  • On import you can merge, replace, or open a new workspace. Duplicate requests can be merged or skipped.

Features

  • Import & sessions: drag-and-drop, session history, duplicate merge/skip
  • Workspaces: multi-tab layouts with .bhhb-workspace.json import/export
  • Table & filtering: column filters, tree view (t), global search (/, Ctrl+F)
  • Analysis: diff, replay, cURL export, inspector, comments (n)
  • More: fast indexing, themes, filtered export, keyboard navigation
